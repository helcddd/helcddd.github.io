在vps上搭建cs服务器

需要东西：
一台vps：之前我使用的是vultr（https://my.vultr.com/），现在换了腾讯云服务器。
ssh管理工具：我用的是MobaXterm，也可以用xshell
为什么要搭建：
因为以后如果要学内网渗透，要用cs，以后要研究免杀，也要这个cs生成木马的。
搭建具体步骤：
1.用MobaXterm成功连接到我的vps，安装的系统是centos7版本的

2.环境配置
yum install net-tools

3.安装msf
curl https://raw.githubusercontent.com/rapid7/metasploit-omnibus/master/config/templates/metasploit-framework-wrappers/msfupdate.erb > msfinstall

chmod 755 msfinstall
./msfinstall

4.安装nmap
yum install nmap -y

5.安装sqlmap
wget https://codeload.github.com/sqlmapproject/sqlmap/legacy.tar.gz/master
tar -zxvf master
cd sqlmap
python sqlmap.py

6.安装java
wget https://repo.huaweicloud.com/java/jdk/8u171-b11/jdk-8u171-linux-x64.tar.gz
mkdir /usr/local/java/
tar -zxvf jdk-8u171-linux-x64.tar.gz -C /usr/local/java/

vim /etc/profile
export JAVA_HOME=/usr/local/java/jdk1.8.0_171
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH

source /etc/profile
ln -s /usr/local/java/jdk1.8.0_171/bin/java /usr/bin/java
java -version

7.安装cs（需要本机有cs的压缩包）
yum install unzip
unzip cobaltstrike4.3.zip
注意要安装java才能运行
还要关防火墙
systemctl stop firewalld.service
systemctl disable firewalld.service

8.成功开启cs
./teamserver ip 随意设置你的密码（例如admin）
./teamserver ip admin
即可开启。
然后在本机连接cs，具体方法可百度或谷歌解决。（到了这里，已经不是问题了，前提是你的cs是可以正常使用的，有些版本有问题）

9.最后
贴个cs的下载链接，群里的，我放到百度云里。
