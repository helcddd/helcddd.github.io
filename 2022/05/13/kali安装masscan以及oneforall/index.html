kali安装github上的工具

前言：
安装github上面的两个工具，分别是masscan和oneforall。
其实我不怎么用，但是有人推荐了，我也去试了下，是挺好用的。
正好，我最近重新搞了下kali，要重新整理工具，也重新梳理下思路。

oneforall：OneForAll是一款功能强大的子域收集工具
masscan：海量 IP 端口扫描器，和nmap类似。

子域名
端口扫描

1.安装oneforall：
再次说下，是在kali上安装。
首先，去到它的项目地址处：https://github.com/shmilylty/OneForAll
只是为了看看它的作用，也防止待会忘了命令。

来到kali的命令终端，cd到桌面，输入：
git clone https://gitee.com/shmilylty/OneForAll.git
从gitee处爬取项目，原因是kali里没挂代理，我这网不行，从github上总是超时。

然后就是输入一些很简单的命令：
cd OneForAll

别忘了安装python3和pip3，如果安装了可以跳过这儿，没有的话。
输入以下命令：
直接输入python3，会提示你如何安装的。
pip3的话，
apt-get install python3-pip

ok了。

现在是位于oneforall的目录下，安装它的依赖。
pip3 install -r requirements.txt

没有报错的话，这样就好了，就可以测试下是否能成功运行了，可以用个域名测试下。

python3 oneforall.py –target xx.com run

然后，结束后，结果会放在项目目录下的results下，是个csv文件。
如果kali里安装了wps的话，那就更方便了。
直接：
open ./results/xx.com.csv
就可以打开了。

至此，oneforall安装完成！

2.安装masscan
还是来到它的github项目地址处：https://github.com/robertdavidgraham/masscan
可以翻译下页面。

同样在kali终端处，回到桌面。
输入：
apt-get –assume-yes install git make gcc
安装gcc，我的话提示我安装好了，没安装gcc的话就让它自己跑。

git clone https://github.com/robertdavidgraham/masscan
然后就出现问题了。可能你也会遇到，一直超时。
没办法，最后我是在本机下好了后，开了python的http服务，再在kali上访问这个http，下载下来的。
其实也很简单。
本机有python3的话，就是cmd输入：
python -m http.server
会在本机的8000端口开http服务。
注意要到那个masscan的目录处进入cmd，后，输入。

kali下好后。
就可以继续了。
cd masscan
make
等待它编译完成。

如果一切顺利的话，此时直接在任意位置，都可以输入:
masscan
会发现能返回它的帮助信息，就说明安装成功了。

此时，可以测试下它。
masscan ip -p 1-8000

就能看到一些端口开放情况了。
至此，masscan安装成功！

3.总结
结束了，成功在kali上安装了oneforall和masscan。
注意一点，以后使用这两个工具，masscan可以直接在任意终端位置使用。
而oneforall必须得到它的目录下去使用。
